# Eloquent
Creating a Movie website to review Laravel ELoquent
# Installation
<ol>
<li>git clone</li>
<li>Composer Install</li>
<li>npm install && npm run dev</li>
<li>php artisan key:generate</li>
<li>php artisan serve</li>
<li>php artisan migrate</li>
</ol>
